# 跨尺度毛细效应多尺度物理信息神经网络求解器

## 项目简介
本项目实现了一种结合坐标变换与垂直边界层展开的多尺度物理信息神经网络（PINN），用于求解二维垂直狭缝中的液膜流动问题。该问题涉及自由边界、表面张力、分子间作用力以及尺度分离现象（垂直厚度远小于水平尺度）。通过渐近分析将解分解为内区主体和边界层修正，并利用PINN分别拟合各尺度成分，实现对宏观界面演化和微观边界层结构的统一求解。
## 物理背景
- 控制方程：不可压缩Navier-Stokes方程 + 分子附加压力项
- 几何：底壁 \(z=0\)，两侧固壁 \(x=\pm H\)，自由界面 \(z=\eta(x,t)\)
- 无量纲小参数：\(\varepsilon = H_0/L \ll 1\)
- 关键无量纲参数：表面张力系数 \(\Gamma\)，分子力参数 \(S\)，重力参数 \(G\)，雷诺数 \(R\)
## 数学方法
1. **坐标变换**：\(y = z/\eta(x,t)\) 将自由边界固定到 \(y\in[0,1]\)
2. **边界层拉伸**：引入 \(y_b = y/\varepsilon\)（底边界层），\(y_t = (1-y)/\varepsilon\)（上边界层）
3. **渐近展开**：将解分解为内区项 + 底边界层项 + 上边界层项，按 \(\varepsilon\) 幂次展开
4. **各阶方程**：得到内区 \(O(1)\) 阶薄膜方程，以及边界层 \(O(\varepsilon^{-1})\) 阶修正方程

## 安装与运行

1.  安装依赖：  
    pip install -r requirements.txt
2.  根据需要修改 config.py 中的参数（如物理常数、网络结构、采样点数等）
3.  运行主程序：  
    python main.py
4.  训练结束后，模型参数保存为 trained\_models.pth，损失曲线保存为 loss\_curve.png
## 关键参数说明
| 参数 | 默认值 | 含义 |
|------|--------|------|
| `EPS` | 0.01 | 小参数 \(\varepsilon\) |
| `GAMMA` | 1.0 | 表面张力系数 |
| `S` | 0.1 | 分子力参数 |
| `L` | 1.0 | 半宽 \(x \in [-L, L]\) |
| `T_MAX` | 1.0 | 最大时间 |
| `YB_MAX` | 10.0 | 底边界层拉伸坐标最大值 |
| `YT_MAX` | 10.0 | 上边界层拉伸坐标最大值 |
| `INNER_LAYERS` | [3,50,50,50,3] | 内区网络结构 |
| `BOTTOM_LAYERS` | [3,30,30,30,3] | 底边界层网络结构 |
| `TOP_LAYERS` | [3,30,30,30,3] | 上边界层网络结构 |
| `ETA_LAYERS` | [2,50,50,50,1] | 界面高度网络结构 |
| `N_INTERIOR` | 5000 | 内区采样点数 |
| `EPOCHS` | 10000 | 训练轮数 |
| `LR` | 1e-3 | 学习率 |
## 自定义修改
- **初始条件**：在 `main.py` 中修改 `my_initial_eta` 函数，或传递给 `trainer.train()`
- **损失权重**：在 `config.py` 中调整 `LAMBDA` 字典
- **网络结构**：修改 `config.py` 中的 `*_LAYERS` 列表
- **物理方程**：若需考虑重力或惯性，设置 `G` 和 `R` 为非零值，并在 `losses.py` 中补充相应项
## 结果与可视化
- 训练过程中每100轮打印当前损失
- 训练结束后生成损失曲线图 `loss_curve.png`
- 可自行扩展代码，在训练后使用模型进行预测并绘制界面演化图、速度场等
## 注意事项
- 确保已安装PyTorch且版本不低于1.10
- 若使用GPU，需安装CUDA版本对应的PyTorch
- 训练时间取决于网络规模和采样点数，建议先用小参数测试
